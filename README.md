# Metalab Karaoke App

## Dependency

- nodejs
- Meteor      
- Meteroite   

``` sh
$ curl https://install.meteor.com | sh      // Install Meteor
$ sudo npm install -g meteorite             // Meteroite
```
## Installing // Setup

``` sh
$ git clone https://github.com/zerocity/karaoke-meteor.git
$ cd karaoke-meteor/
$ mrt 
```
